Despite being small, Red Dwarf is very capable.

You can develop almost any website with it!

Red Dwarf is the ideal tool to develop capable websites in no time.

However, it is strongly opinionated.

  • No JSON response
  • No WebSockets
  • No compression
  • No templates
  • No error handlers
  • No typed parameters
  • No telemetry
  • No TLS
  • No CORS
  • ...
